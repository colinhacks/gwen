# Installation

To install the latest version:

```sh
npm install --save gwen
```

```sh
yarn add gwen
```

#### TypeScript versions

gwen@0.x is compatible with TypeScript 3.7+.

You must use `"strict": true` in your `tsconfig.json`! Otherwise you may get unexpected type errors.
